1. Problem Statement

Many people want a simple and offline method to keep track of their daily health metrics such as weight, steps walked, sleep duration, and personal notes.
Most digital health apps are complex, require internet connectivity, or collect user data.
There is a need for a lightweight, privacy-friendly, and easy-to-use tool that helps users record and monitor their health metrics consistently.

This project solves that problem by providing a simple Python-based console application that stores all data locally in a text file and offers essential tracking features.


---

2. Scope of the Project

The scope of this project includes:

Creating a menu-driven console program in Python.

Enabling users to add health entries including date, weight, steps, sleep hours, and notes.

Displaying all logged entries in a readable format.

Allowing users to delete an entry by selecting its number.

Processing stored data to compute basic statistics such as average weight and average sleep.

Ensuring all data is stored in a text file for easy access and modification.


The project does not include advanced UI, cloud storage, or analytics beyond basic averages.


---

3. Target Users

Students learning programming and wanting a practical Python project.

Individuals who prefer an offline and private method for tracking health.

Beginners who want a simple tool without complex installations or databases.

Users who record health metrics daily and need a minimalistic tracking system.



---

4. High-Level Features

Add Health Entry:
Users can input date, weight, steps, sleep hours, and notes.

View All Entries:
Displays all recorded data from the text file with numbering.

Delete Entry:
Users can remove any entry by selecting its index number.

Statistics Module:
Calculates and displays average weight and average sleep hours.

Persistent Storage:
All entries stored in health.txt for easy readability and offline use.

Input Validation:
Ensures valid date format and appropriate numeric values where required.
